## 1. 회원가입 및 온보딩 플로우

### 1.1 전체 플로우

```
[시작 화면]
    ↓
[회원가입 / 로그인] (Supabase Auth)
    ↓
[온보딩 Step 1] 임신 여부 선택
    ├─ 임신 중 → [Step 2-A] 출산예정일 입력
    └─ 육아 중 → [Step 2-B] 자녀 생년월일 입력
    ↓
[온보딩 Step 3] 직장 여부 선택
    ↓
[온보딩 Step 4] 지역 선택
    ↓
[온보딩 완료] 프로필 저장 + 타임라인 자동 생성
    ↓
[홈 화면] (첫 방문)
```

---

### 1.2 화면별 상세

#### 시작 화면
**목적**: 앱 소개 및 회원가입/로그인 유도

**구성 요소**:
- 앱 로고 + "BebeCare"
- 슬로건: "임신부터 육아까지, AI가 맞춤 조언"
- 주요 기능 소개 (3개 슬라이드)
  - Slide 1: "개인화된 AI 조언"
  - Slide 2: "놓치지 않는 타임라인"
  - Slide 3: "정부 지원금 한눈에"
- 버튼:
  - "회원가입" (Primary, 큰 버튼)
  - "이미 계정이 있어요" (Secondary, 작은 텍스트)

---

#### 회원가입 / 로그인
**목적**: Supabase Auth를 통한 인증

**구성 요소**:
- 이메일 + 비밀번호 입력
- "회원가입" 버튼
- 소셜 로그인 (선택사항):
  - Google 로그인
  - Apple 로그인 (iOS)
- "로그인으로 돌아가기" 링크

**유효성 검사**:
- 이메일 형식 확인
- 비밀번호 최소 8자 이상

---

#### 온보딩 Step 1: 임신 여부 선택
**목적**: 사용자 유형 파악 (임신부 vs 육아 부모)

**구성 요소**:
- 제목: "어떤 정보를 받고 싶으세요?"
- 버튼 2개 (큰 버튼, 아이콘 포함):
  - "임신 중" (👶 아이콘)
  - "육아 중" (🍼 아이콘)
- 진행률 표시: 1/4

**선택 시 액션**:
- "임신 중" → Step 2-A로 이동
- "육아 중" → Step 2-B로 이동

---

#### 온보딩 Step 2-A: 출산예정일 입력 (임신부)
**목적**: 임신 주차 계산 및 타임라인 생성 기준

**구성 요소**:
- 제목: "출산예정일을 알려주세요"
- 설명: "정확한 주차별 정보를 제공해드려요"
- 달력 UI (Date Picker)
  - 오늘 이후 날짜만 선택 가능
  - 기본값: 오늘 + 280일 (40주)
- 계산된 임신 주차 표시: "현재 28주차입니다"
- 진행률 표시: 2/4
- 버튼: "다음"

---

#### 온보딩 Step 2-B: 자녀 생년월일 입력 (육아 부모)
**목적**: 아이 개월 수 계산 및 타임라인 생성 기준

**구성 요소**:
- 제목: "자녀 생년월일을 알려주세요"
- 설명: "개월별 맞춤 정보를 제공해드려요"
- 자녀 정보 입력:
  - 이름 (선택사항)
  - 생년월일 (필수, Date Picker, 오늘 이전만 선택 가능)
  - 성별 (선택사항): 남 / 여 / 선택 안 함
- "+ 자녀 추가" 버튼 (최대 5명)
- 계산된 개월 수 표시: "생후 3개월입니다"
- 진행률 표시: 2/4
- 버튼: "다음"

---

#### 온보딩 Step 3: 직장 여부 선택
**목적**: 직장인 맞춤 정보 제공 (출산휴가, 육아휴직 등)

**구성 요소**:
- 제목: "현재 직장에 다니고 계신가요?"
- 버튼 2개:
  - "네, 근무 중이에요" (💼 아이콘)
  - "아니요, 비근무 중이에요" (🏠 아이콘)
- 진행률 표시: 3/4
- 버튼: "다음"

---

#### 온보딩 Step 4: 지역 선택
**목적**: 지역별 정부 지원금, 어린이집 정보 제공

**구성 요소**:
- 제목: "거주하시는 지역을 알려주세요"
- 설명: "지역별 지원금 정보를 제공해드려요"
- 드롭다운 2개:
  - 시·도 선택 (서울, 부산, 경기 등)
  - 시·군·구 선택 (강남구, 해운대구 등)
    - 시·도 선택 시 동적으로 옵션 변경
- 진행률 표시: 4/4
- 버튼: "완료"

---

#### 온보딩 완료 화면
**목적**: 프로필 저장 및 타임라인 자동 생성

**구성 요소**:
- 체크 아이콘 애니메이션 (✅)
- 제목: "프로필 설정 완료!"
- 설명: "맞춤 정보를 준비하고 있어요..."
- 로딩 스피너 (타임라인 자동 생성 중)
- 자동 이동 (3초 후): 홈 화면

**백그라운드 작업**:
- Supabase에 프로필 저장
- 타임라인 자동 생성 API 호출
- 홈 화면 데이터 미리 로드

## 2. 홈 화면 구성

### 2.1 화면 레이아웃

```
[헤더]
  - 로고 + 사용자 이름
  - 설정 버튼 (⚙️)

[맞춤 조언 섹션]
  - 제목: "지금 알아야 할 정보"
  - AI 조언 카드 2-3개 (가로 스크롤)

[타임라인 섹션]
  - 제목: "이번 주 할 일" + 완료율 (3/5)
  - 체크리스트 항목 3-5개

[추천 콘텐츠 섹션] (선택사항, MVP 축소)
  - 제목: "추천 정보"
  - 카테고리별 콘텐츠 카드

[하단 네비게이션]
  - 홈 (🏠, 활성)
  - 타임라인 (📅)
  - AI 챗봇 (💬)
  - 설정 (⚙️)
```

---

### 2.2 주요 컴포넌트

#### 맞춤 조언 카드
**디자인**:
- 카드 크기: 300px × 150px (가로 스크롤)
- 배경색: 카테고리별 (건강: 파랑, 보험: 주황, 직장: 초록)
- 아이콘 + 제목 + 간단한 설명
- 하단 버튼: "자세히 보기" 또는 "AI에게 물어보기"

**예시**:
```
┌─────────────────────────┐
│ 💙 건강                 │
│ 28주차 체크리스트        │
│                         │
│ - 임신 당뇨 검사        │
│ - 태아보험 마감 임박    │
│                         │
│ [AI에게 물어보기 →]     │
└─────────────────────────┘
```

---

#### 타임라인 체크리스트 항목
**디자인**:
- 체크박스 + 제목 + 일정 날짜
- 우선순위 색상 (높음: 🔴, 중간: 🟠, 낮음: ⚪)
- 카테고리 아이콘 (건강: 💊, 보험: 🛡️, 직장: 💼)

**예시**:
```
☐ 🔴 💊 임신 당뇨 검사 (2월 15일)
☑️ 🟠 🛡️ 태아보험 상담 예약 (2월 10일, 완료)
☐ ⚪ 📋 출산 준비물 리스트 (2월 20일)
```

## 3. AI 챗봇 플로우

### 3.1 전체 플로우

```
[홈 화면]
    ↓
[하단 네비게이션 > AI 챗봇 버튼] 클릭
    ↓
[AI 챗봇 화면]
    ↓
[사용자 질문 입력] → [전송]
    ↓
[AI 응답 대기] (로딩 애니메이션 "...")
    ↓
[AI 답변 표시]
    ↓
[추가 질문] 또는 [홈으로 돌아가기]
```

---

### 3.2 화면 구성

#### AI 챗봇 화면
**레이아웃**:
- 헤더: "AI 조언"
- 메시지 리스트 (스크롤 가능)
  - 사용자 메시지 (오른쪽, 파란색 버블)
  - AI 메시지 (왼쪽, 회색 버블)
- 입력창 (하단 고정):
  - 텍스트 입력 필드
  - 전송 버튼 (📤)

**예시 대화**:
```
[AI] 안녕하세요! BebeCare AI 어시스턴트입니다.
     임신·육아에 대해 궁금한 점을 물어보세요.

[사용자] 출산휴가는 언제부터 쓸 수 있나요?

[AI] 출산예정일이 2026년 6월 15일이시네요.
     출산휴가는 출산예정일 45일 전부터...
     (상세 답변)

[사용자] 태아보험은 언제 가입해야 하나요?

[AI] 태아보험은 임신 22주 전에 가입하시는 것을
     권장드립니다. 현재 28주차이시므로...
```

---

### 3.3 주요 기능

#### 제안 질문 (Quick Replies)
**목적**: 첫 방문 사용자에게 질문 예시 제공

**구성**:
- AI 첫 메시지 하단에 버튼 3-4개
- 클릭 시 자동으로 질문 전송

**예시**:
```
[AI] 안녕하세요! 궁금한 점을 물어보세요.

[출산휴가 계산] [태아보험 가입 시기] [예방접종 일정]
```

---

#### 대화 히스토리 불러오기
**목적**: 이전 대화 내용 참고

**구성**:
- 챗봇 화면 진입 시 최근 10개 메시지 자동 로드
- 위로 스크롤 시 추가 메시지 로드 (무한 스크롤)

## 4. 타임라인 관리 플로우

### 4.1 전체 플로우

```
[홈 화면] 타임라인 섹션
    ↓
[체크박스 클릭] → 완료/미완료 토글 (즉시 반영)
    ↓
[타임라인 항목 클릭] → [타임라인 상세 화면]
    ↓
[액션 선택]
    ├─ [완료 처리]
    ├─ [일정 수정] (선택사항)
    └─ [AI에게 물어보기] → AI 챗봇 화면 이동
```

---

### 4.2 화면 구성

#### 타임라인 전체 화면
**레이아웃** (하단 네비게이션 > 타임라인):
- 탭 3개:
  - "이번 주" (활성)
  - "다음 주"
  - "전체"
- 타임라인 리스트 (그룹화):
  - 날짜별 그룹
  - 각 그룹 내 체크리스트 항목

**예시**:
```
┌─────────────────────────┐
│ 이번 주 | 다음 주 | 전체 │
└─────────────────────────┘

2월 15일 (금)
☐ 🔴 💊 임신 당뇨 검사
☐ 🟠 📋 출산 준비물 리스트 작성

2월 17일 (일)
☑️ 🟠 🛡️ 태아보험 상담 예약 (완료)

2월 20일 (수)
☐ ⚪ 📦 신생아 용품 구매
```

---

#### 타임라인 상세 화면
**레이아웃**:
- 제목
- 일정 날짜
- 카테고리 + 우선순위
- 설명 (상세 정보)
- 완료 여부 토글
- 버튼:
  - "AI에게 물어보기" → 관련 질문으로 챗봇 이동

**예시**:
```
┌─────────────────────────┐
│ 💊 건강 | 높음 🔴       │
│ 임신 당뇨 검사          │
│                         │
│ 📅 2월 15일 (금)       │
│                         │
│ 24-28주 사이에 임신    │
│ 당뇨 검사를 받으세요.  │
│ 공복 혈당 측정 후...   │
│                         │
│ [✅ 완료 처리]          │
│ [AI에게 물어보기 →]    │
└─────────────────────────┘
```

---

## 5. 프로필 수정 플로우

### 5.1 전체 플로우

```
[홈 화면]
    ↓
[하단 네비게이션 > 설정] 클릭
    ↓
[설정 화면] → [프로필 수정] 메뉴 클릭
    ↓
[프로필 수정 화면]
    ↓
[정보 수정] (임신 정보, 직장, 지역, 자녀 등)
    ↓
[저장] → 타임라인 자동 업데이트
    ↓
[설정 화면으로 돌아가기]
```

---

### 5.2 화면 구성

#### 프로필 수정 화면
**레이아웃**:
- 섹션별 그룹화:
  - **임신 정보** (임신부만):
    - 출산예정일 (Date Picker)
    - 계산된 임신 주차 표시
  - **직장 정보**:
    - 직장 여부 (토글 스위치)
  - **지역 정보**:
    - 시·도 (드롭다운)
    - 시·군·구 (드롭다운)
  - **자녀 정보** (육아 부모만):
    - 자녀 카드 리스트
    - "+ 자녀 추가" 버튼
- 하단 버튼: "저장"

---

#### 자녀 관리
**카드 디자인**:
- 자녀 이름 + 생년월일 + 성별
- 계산된 개월 수 표시
- 수정 버튼 (✏️)
- 삭제 버튼 (🗑️)

**예시**:
```
┌─────────────────────────┐
│ 👶 민준 (남)            │
│ 2025-11-07 (생후 3개월)│
│                         │
│ [✏️ 수정] [🗑️ 삭제]    │
└─────────────────────────┘
```

---

## 6. 설정 및 기타

### 6.1 설정 화면
**메뉴 구성**:
- **계정**:
  - 프로필 수정
  - 비밀번호 변경
  - 로그아웃
- **알림**:
  - 푸시 알림 ON/OFF
  - 알림 시간대 설정 (오전 9시, 오후 6시 등)
- **일반**:
  - 앱 버전
  - 이용약관
  - 개인정보 처리방침

---

### 6.2 주요 사용자 여정 요약

#### 임신부 여정
```
회원가입 → 온보딩 (출산예정일 입력)
  ↓
홈 화면 (맞춤 조언 + 타임라인)
  ↓
타임라인 확인 (22주 태아보험, 34주 출산휴가)
  ↓
AI 챗봇 ("출산휴가는 언제부터?")
  ↓
체크리스트 완료 (태아보험 상담 예약 완료)
```

#### 신생아 부모 여정
```
회원가입 → 온보딩 (자녀 생년월일 입력)
  ↓
홈 화면 (맞춤 조언 + 타임라인)
  ↓

  ↓

```

## 7. UX 원칙

### 7.1 디자인 원칙
1. **모바일 우선**: 스마트폰에서 사용하기 쉬운 큰 버튼, 터치 친화적 UI
2. **명확한 정보 계층**: 중요한 정보를 먼저, 부가 정보는 접어두기
3. **일관된 디자인 시스템**: 색상, 타이포그래피, 아이콘 일관성
4. **접근성**: WCAG 2.1 AA 준수, 스크린 리더 지원

---

### 7.2 사용자 경험 원칙
1. **빠른 온보딩**: 4단계 이내, 3분 이내 완료
2. **즉각적인 피드백**: 버튼 클릭, 데이터 저장 시 즉시 피드백
3. **오류 방지**: 유효성 검사, 명확한 에러 메시지
4. **진행 상황 표시**: 로딩 스피너, 진행률 표시
